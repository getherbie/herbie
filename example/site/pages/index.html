Welcome to Herbie

With Herbie you are able to create a fully functional website or blog in a short amount of time and with little effort. Herbie is based on proven libraries and concepts and supports several PHP Standards Recommendations. It uses Twig as Templating Engine and offers a huge amount of variables, functions, filters and tests. It comes with an event and plugin manager based on Zend\EventManager to ensure extensibility and portability. And it offers some really useful plugins all managed by Composer & Packagist.

Hungry for more? Give it a try!
